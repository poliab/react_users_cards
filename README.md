# Users List

Небольшое учебное React-приложение для практики **props**, рендеринга списков и условного отображения данных. В проекте выводится список пользователей в виде карточек с аватаром, именем, возрастом и email.

## Что реализовано

- Передача данных из родительского компонента в дочерние через props.
- Рендеринг списка пользователей через `map()` с уникальным `key`.
- Отдельный компонент `UserCard` для отображения одной карточки пользователя.
- Условный fallback для пользователя без аватарки.
- Простые стили карточек: тень, скругление, hover-эффект с лёгким поднятием карточки вверх через `transform: translateY(...)`.

## Структура проекта

```bash
src/
├── components/
│   ├── UserCard.tsx
│   └── Users.tsx
├── App.tsx
└── App.css
```

## Как работает приложение

Компонент `App` отображает заголовок страницы, компонент `Users` и footer. Компонент `Users` хранит массив пользователей и передаёт данные в `UserCard`. Каждый `UserCard` получает props одного пользователя и показывает его на странице.

## Технологии

- React
- TypeScript
- CSS
- Vite

## Запуск проекта

1. Установить зависимости:

```bash
npm install
```

2. Запустить проект:

```bash
npm run dev
```

3. Открыть приложение в браузере по адресу, который покажет Vite, обычно это `http://localhost:5173`.

## Чему учит этот проект

- Понимать, как работают props в React.
- Разделять приложение на маленькие переиспользуемые компоненты.
- Передавать данные сверху вниз: `App -> Users -> UserCard`.
- Работать с простым условным рендерингом в JSX.
