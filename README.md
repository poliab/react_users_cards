# Users List

Невеликий навчальний React‑застосунок для практики **props**, рендерингу списків та умовного відображення даних. У проєкті виводиться список користувачів у вигляді карток з аватаром, ім’ям, віком та email.

## Що реалізовано

- Передавання даних з батьківського компонента в дочірні за допомогою props.
- Рендеринг списку користувачів через `map()` з унікальним `key`.
- Окремий компонент `UserCard` для відображення однієї картки користувача.
- Умовний fallback для користувача без аватарки.
- Прості стилі карток: тінь, скруглення, hover‑ефект із легким підняттям картки вгору через `transform: translateY(...)`.

## Структура проєкту

```bash
src/
├── components/
│   ├── UserCard.tsx
│   └── Users.tsx
├── App.tsx
└── App.css
```

Як працює застосунок

Компонент  App  відображає заголовок сторінки, компонент  Users  та footer.
Компонент  Users  зберігає масив користувачів і передає дані в  UserCard .
Кожен  UserCard  отримує props одного користувача і показує його на сторінці.

Запуск проєкту

	1.	Встановити залежності:
npm install
	2.	Запустити проєкт:
npm run dev
	3.	Відкрити застосунок у браузері за адресою, яку покаже Vite (зазвичай це  http://localhost:5173 ).

Чого навчає цей проєкт

	•	Розуміти, як працюють props у React.
	•	Ділити застосунок на маленькі багаторазово придатні компоненти.
	•	Передавати дані згори донизу:  App -> Users -> UserCard .
	•	Працювати з простим умовним рендерингом у JSX.
